<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick Start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with the Sinapsis API in minutes.
  </p>

  <Steps>
    <Step title="Set up your environment">
      Install dependencies and configure your local environment with Docker and Bun.
      
      ```bash
      git clone https://github.com/JonathanLara27/backend-sinapsis-test
      cd backend-sinapsis-test
      bun install
      ```
      
      See the [Installation guide](/installation) for detailed setup instructions.
    </Step>

    <Step title="Start the database">
      Launch MySQL using Docker Compose to run the database locally.
      
      ```bash
      docker-compose up -d
      ```
      
      The database will be available on port 3307 to avoid conflicts with local MySQL installations.
    </Step>

    <Step title="Run the API">
      Start the development server and access the Swagger documentation.
      
      ```bash
      bun run start:dev
      ```
      
      <Note>
        Access the interactive API documentation at `http://localhost:3000/dev/api/docs`
      </Note>
    </Step>

    <Step title="Make your first API call">
      Create a campaign with messages using the REST API.
      
      ```bash
      curl -X POST http://localhost:3000/dev/api/campaigns \
        -H "Content-Type: application/json" \
        -d '{
          "name": "Summer Sale Campaign",
          "scheduledAt": "2024-07-01T10:00:00Z",
          "userId": 1,
          "messages": ["Special offer just for you!", "Limited time only!"]
        }'
      ```
      
      Check out the [API Reference](/api/overview) for all available endpoints.
    </Step>
  </Steps>
</div>
