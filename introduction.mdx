# Sinapsis Marketing Campaigns API

A production-ready RESTful API for managing and scheduling mass marketing campaigns, built with **NestJS** and designed for serverless deployment on AWS Lambda.

## What It Does

The Sinapsis Marketing Campaigns API provides a complete solution for:

- **Campaign Management**: Create and schedule marketing campaigns with multiple messages
- **User & Client Management**: Full CRUD operations with pagination and soft delete capabilities
- **Advanced Reporting**: Generate dashboard reports showing message delivery status with optimized SQL queries
- **Transactional Integrity**: ACID transactions ensure campaigns and messages are created atomically with automatic rollback on failures

## Technology Stack

The API is built on modern, production-proven technologies:

### Core Framework
- **NestJS 11.x** - Progressive Node.js framework with TypeScript support
- **TypeScript 5.7** - Type-safe development with latest language features
- **Bun Runtime** - Fast JavaScript runtime for development

### Database Layer
- **TypeORM 0.3** - Powerful ORM with QueryBuilder and transaction support

## Architecture Highlights

### Repository Pattern with QueryBuilder
The reports module uses TypeORM's QueryBuilder to delegate aggregation and counting operations to the database engine, optimizing performance for large datasets:

```typescript
// Efficient query that processes data at the database level
GET /api/reports/dashboard?month=2026-03&clientId=1
```

### Manual Transaction Control
Campaign creation uses manual `QueryRunner` transactions to ensure atomicity:

```typescript
POST /api/campaigns
{
  "name": "Campaña Navidad 2026",
  "scheduledAt": "2026-12-24T10:00:00Z",
  "userId": 1,
  "messages": ["Oferta Exclusiva", "Descuento 50%"]
}
```

If any message fails to save, the entire campaign is rolled back automatically.
